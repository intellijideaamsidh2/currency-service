Open Built Docs How to build

Build the docs (one-time)

  1. Install Python (if not already installed)
  2. Install MkDocs dependencies
  3. Build the static site into mkdocs/site

Run these in Command Prompt from C:\Users\amsid\currency-service: 

python -m pip install -r mkdocs\requirements.txt
mkdocs build -f mkdocs\mkdocs.yml

After build, click "Open Built Docs" above.
